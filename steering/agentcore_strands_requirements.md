# Required Modules for Building Agentic Applications

## Strands and AgentCore Requirements

When building agentic applications using Strands Agents and Amazon Bedrock AgentCore, include these modules in your `requirements.txt`:

```
strands-agents
strands-agents-tools
boto3
bedrock-agentcore
bedrock-agentcore-starter-toolkit
```

## Module Descriptions

- **strands-agents**: Core Strands Agent SDK for building agentic applications
- **strands-agents-tools**: Additional tools and utilities for Strands agents
- **boto3**: AWS SDK for Python to interact with AWS services
- **bedrock-agentcore**: Amazon Bedrock AgentCore SDK for production-ready agent deployment
- **bedrock-agentcore-starter-toolkit**: CLI toolkit and starter templates for AgentCore
